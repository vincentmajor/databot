databot

A very simple slack-based bot that listens to conversations that contain the word 'data'.

Created in Node.js using a botkit template from Beep Boop.

Usage

When databot is invited to a channel, it will ambiently listen until it hears case-insensitive 'data'. It does not need to be bound by whitespace so 'database' and 'dataset' will trigger action. databot's only response is:

Did someone say data?

Run in Beep Boop

This repository is connected to Beep Boop so any commits here will automatically deploy to my team.
